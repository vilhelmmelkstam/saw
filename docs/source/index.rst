SAW, a SimFin API Wrapper
=========================

SAW is a API wrapper for the great `SimFin API <https://simfin.com/api/v1/documentation/>`_.

SimFin is a great tool to access financial data for for american companies for free. In august 2018 SimFin added an API to access their data and this package is a wrapper in Python around that API. 

Installation
^^^^^^^^^^^^
SAW can be installed using pip::

	pip install saw
